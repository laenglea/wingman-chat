### Vision helper (`vision`)

Both the Python and JavaScript interpreters expose a `vision` helper that sends an image to a vision-capable language model and returns its text response. Use it to describe photos, read charts and diagrams, or answer questions about screenshots.

- Python: `text = await vision("/home/user/photo.png")` — async, returns `str`; an optional second argument is a custom prompt: `await vision("/home/user/chart.png", "List the data points as CSV")`
- JavaScript: `const text = await vision("/photo.png")` — async, returns a string; an optional second argument is a custom prompt: `await vision("/chart.png", "List the data points as CSV")`

Use this helper for workspace-only images or programmatic pipelines. Inspect images included in the
user's message with built-in vision instead. Without a prompt the helper transcribes text or describes
the image. For precise transcription use `ocr`; use `vision` for image understanding or targeted
questions. One call per image — it is a network round trip.
