## Visual Style — Bricks

Create a bold, brick-stacked block infographic:
- Strong, saturated color palette: each section uses a distinct bold color block — cobalt blue (#1B4DFF), crimson (#DC2626), emerald (#059669), amber (#D97706), purple (#7C3AED), charcoal (#374151)
- Layout built from stacked rectangular blocks/bricks of different widths and heights, tightly packed like a masonry wall
- Each brick contains a self-contained piece of content: a heading, a stat, a mini-chart, or an icon with label
- Bold, high-contrast white or light text on colored backgrounds
- Strong geometric shapes — sharp corners or minimal rounding, solid fills, no gradients
- Thick borders or gaps between bricks creating a clear modular structure
- Large bold sans-serif typography — condensed or uppercase for impact
- Data visualizations integrated into bricks: progress bars, filled blocks, stacked segments
- No decorative elements — pure content density with color doing the visual work
- Overall feel: bold, structured, high-impact — like LEGO blocks, a colored brick wall, or a bold data dashboard
